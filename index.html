Projects (Data Science)

Picked For Me

This project uses NLP models to help picky eaters discover diverse recipes.

  • Preprocessed 15K+ recipes from food.com (dataset from kaggle.com) with Pandas
  • Used NLP methods for comparisons - recipe to recipe and ingredient to ingredient - with gensim and scikit-learn
  • Deployed application using Flask, AWS, and Oracle so the user can search for diverse recipes by specifying likeness of foods
  • GitHub Repository | GitHub Repository - App | Deployment

Predicting the Functionality of Water Wells

This project uses Machine Learning models to predict if a water well in Tanzania is in need of repair.

  • Performed a thorough EDA of the dataset “Pump It Up - Data Mining the Water Table” with ~60,000 records (water wells)
  • Built a data pipeline to extract features for 5 Machine Learning models to classify each well by repair need status
  • Utilized hyperparameter tuning to identify a Random Forest Classifier with the best performance metrics
  • GitHub Repository

Ensuring a Better Future

This project offers an updated estimate of the number of Opportunity Youth (OY) in South King County, WA.

  • Queried the American Community Survey Public Use MicroData 5-year via SQL
  • Updated the number of OY from 19,000 in 2016 and compared changes in the total numbers for 3 sub-age groups
  • Identified key volume change in the oldest age group and noted additional data points to collect
  • GitHub Repository
